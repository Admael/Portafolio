Propiedad outline

Outline dotted: Define un contorno punteado. Width: thick(5px)

Outline dashed: Define un contorno discontinuo. Width: thick

Outline solid: Define un contorno sólido. Width: medium (3px)

Outline double: Define un doble contorno. Width: medium

Outline groove: Define un contorno ranurado en 3D. Width: thin(1px)

Outline ridge: Define un contorno estriado en 3D. Width: thin

Outline inset: Define un contorno de inserción (inferior) en 3D. Width: 5px

Outline outset: Define un contorno inicial (superior) en 3D. Width: 6px

Outline none: No define contorno.

Outline hidden: Define un contorno oculto.

Outline -> Propiedad abreviada para establecer las siguientes propiedades de contorno individuales: outline-width | outline-style | outline-color
